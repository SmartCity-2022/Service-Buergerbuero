# Service-Buergerbuero

__Microservice Buergerbuero__

Dieser Microservice stellt Informationen und Dienstleistungen des Bürgerbüros zur Verfügung:  
- Melden von Umzügen
- Terminvereinbarung
- Sperrmüll Anmeldung
- Fundsachen Einsicht
